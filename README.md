# Analisis_de_Desempeno_Sanitario_durante_COVID19_en_Argentina
Este proyecto desarrolla un marco analítico reproducible para evaluar el desempeño de los sistemas hospitalarios provinciales argentinos durante la pandemia de COVID-19 (marzo 2020 – junio 2022), utilizando registros oficiales del SISA (Ministerio de Salud de la Nación).
